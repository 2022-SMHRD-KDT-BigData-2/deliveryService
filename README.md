## 우리할수있조!? 프로젝트명
- 대학생 맞춤형 심부름 서비스

## 주제 선정
- 비대면 서비스의 수요 증가에 따른 심부름 서비스 활성화

## 주요기능
- 심부름 게시글 작성 및 신청
- 물물교환 게시글 작성 및 신청
- 회원관리 기능(로그인, 회원가입, 회원정보 수정 및 탈퇴, 이용내역 확인, 헬퍼 인증)
- 카카오맵 API를 활용한 위치 활성화
- 고객센터 게시판

## 개발환경
- 사용언어 : Java 1.8 / HTML / CSS / Javascript
- 개발도구 : eclipse / VSCode
- 서버환경 : tomcat 8.5 / Flask
- 데이터베이스 : oracle 11g
- 빌드도구 : maven
